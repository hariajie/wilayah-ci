###################
Dropdown Wilayah Indonesia - Codeigniter
###################

Script untuk membuat dropdown pilihan wilayah di Indonesia dari Provinsi hingga Kelurahan dengan lebih mudah.  Script dibuat menggunakan Framework Codeigniter.


*******************
Versi
*******************

- PHP 5.6.30
- MySQL 5.0.11
- Codeigniter 3.1.11
- JQuery 3.5.0
- Bootstrap 4.4.1
